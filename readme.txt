== Description ==

A WordPress plugin that pulls in Instagram images of user or hashtags.Display your latest photos, tagged photos, photos from a place or location, your favourite photos, your feed, or other users photos. Our widget can be set up to fit any website or blog in three ways - with a Grid And CSS3 Zoom Effect.

Features:

* Images from instagram are imported as wordpress attachments
* Display Images With CSS3 Zoom Effect
* No Api Key Needed
* Link images to user profile, image url, locally saved image, attachment url, custom url or none
* For more info visit http://freepiratemovie.com/


[a plugin by Scott Evans](http://scott.ee/ "WordPress designer and developer")

== Installation ==

To install this plugin:

1. Upload the `instagram-zoom-widget` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. That's it!
